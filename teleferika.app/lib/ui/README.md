# UI

This directory contains all user interface components.

## Structure

- `screens/` - Full-page screens that users navigate to, organized by feature
  - `ble/` - BLE/RTK device selection and configuration
  - `loading/` - Initial loading and migration
  - `map/` - Map view screen
  - `points/` - Points list and point editor
  - `projects/` - Projects list and project tabbed view
  - `settings/` - App settings
- `widgets/` - Reusable UI components shared across screens
