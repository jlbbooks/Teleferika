Export project data using licensed export service

Exports project data in the specified format using the premium export functionality.

Parameters

  • format: The export format (e.g., 'kml', 'csv', 'geojson')
  • project: The project data to export
  • points: List of points to include in the export

Returns

true if export was successful, false otherwise

final success = await LicensedFeaturesLoader.exportProject(
  'kml',
  projectData,
  pointList,
);

Implementation

static Future<bool> exportProject(
  String format,
  dynamic project,
  List<dynamic> points,
) async {
  try {
    final result = await FeatureRegistry.executeFunction(
      _pluginName,
      'export_project',
      {'format': format, 'project': project, 'points': points},
    );
    return result as bool? ?? false;
  } catch (e) {
    logger.severe('Error exporting project', e);
    return false;
  }
}
