Show map download page

Navigates to the map download page where users can select areas and download offline map tiles.

Parameters

  • context: The build context for navigation

Returns

true if navigation was successful, false otherwise

Example

final success = await LicensedFeaturesLoader.showMapDownloadPage(context);
if (success) {
  print('Map download page opened');
}

Implementation

static Future<bool> showMapDownloadPage(BuildContext context) async {
  try {
    final result = await FeatureRegistry.executeFunction(
      _pluginName,
      'show_map_download_page',
      {'context': context},
    );
    return result as bool? ?? false;
  } catch (e) {
    logger.severe('Error showing map download page', e);
    return false;
  }
}
