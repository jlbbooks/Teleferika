Map<String, dynamic>? getLicenceStatus()

Get licence status for feature availability

Returns detailed information about the current licence including expiry date, feature list, and status.

Returns

Map containing licence information or null if unavailable

Example

final status = LicensedFeaturesLoader.getLicenceStatus();
if (status != null) {
  print('Licence expires: ${status['expiry_date']}');
}

Implementation

static Map<String, dynamic>? getLicenceStatus() {
  try {
    final result = FeatureRegistry.executeFunction(
      _pluginName,
      'get_licence_info',
    );
    return result as Map<String, dynamic>?;
  } catch (e) {
    logger.severe('Error getting licence status', e);
    return null;
  }
}
