Show export dialog with format selection

Displays a dialog allowing users to select export format and save location for project data.

Parameters

  • context: The build context for showing the dialog
  • project: The project data to export
  • points: List of points to include in the export
  • onExportComplete: Optional callback when export completes

Returns

true if export was initiated successfully, false otherwise

Example

final success = await LicensedFeaturesLoader.showExportDialog(
  context,
  projectData,
  pointList,
  onExportComplete: (success) {
    print('Export completed: $success');
  },
);

Implementation

static Future<bool> showExportDialog(
  BuildContext context,
  dynamic project,
  List<dynamic> points, {
  Function(bool)? onExportComplete,
}) async {
  try {
    final result = await FeatureRegistry.executeFunction(
      _pluginName,
      'show_export_dialog',
      {
        'context': context,
        'project': project,
        'points': points,
        'on_export_complete': onExportComplete,
      },
    );
    return result as bool? ?? false;
  } catch (e) {
    logger.severe('Error showing export dialog', e);
    return false;
  }
}
