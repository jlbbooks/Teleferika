Creates a success status with green styling and check icon.

Success messages are typically shown after successful operations like saving files, completing forms, or successful API calls.

Auto-hide duration: 3 seconds Color: Green Icon: Check circle

Example:

StatusIndicator(
  status: StatusManager.success('Project saved successfully!'),
)

Implementation

static StatusInfo success(String message) => StatusInfo(
  type: StatusType.success,
  message: message,
  icon: Icons.check_circle,
  color: Colors.green,
);
